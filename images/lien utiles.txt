## 📦 Uber Eats — Source de vérité pour le menu
URL : https://www.ubereats.com/fr/store/dr-wings-franconville/bbcpgSVOWPm8s-LtDvh50Q

Quoi faire :
- Fetche cette page via Firecrawl UNIQUEMENT (Uber Eats est une SPA avec JS dynamique — le fetch natif ne fonctionnera pas)
- Utilise firecrawl_scrape avec rendu JS activé
- Récupère TOUT le menu : catégories dans l'ordre d'affichage, noms des plats, descriptions, prix
- Récupère l'image individuelle de chaque plat — c'est la priorité absolue
- Si un plat n'a pas d'image sur Uber Eats, utilise un placeholder stylé à l'identité Dr Wings
- Pour les descriptions : si Uber Eats en fournit une, utilise-la. Si le plat n'en a pas mais que son nom est suffisamment explicite (ex : "Korean Wings sauce soja sucrée"), génère une description courte et dans le ton Dr Wings. Si le nom est trop vague ou trop signature pour inventer (ex : "Sauce spéciale du docteur"), laisse sans description.
- Récupère les avis clients disponibles sur la page
- Récupère les infos du restaurant : horaires, adresse, note, popularité des plats
